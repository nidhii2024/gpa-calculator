# gpa-calculator
Semester gpa calculator for students

HTML - Structure of the page
CSS - All the styling, colors, animations, gradients
JavaScript - All the calculations, logic, and interactivity


How It Works:
Everything happens locally in the browser:

You enter data → JavaScript stores it in memory
You click Calculate → JavaScript does math (Cartesian product of grade combinations)
Results appear → JavaScript generates HTML dynamically and shows it
